Wunderkind jQuery Plugin

Wunderkind is a jQuery plugin that generates CSS imagemaps (non-spritebased) with hover animations.

Usage

Call wunderkind on a jQuery collection of block level elements containing a single image tag. The image tag will serve as the default (off) state for the wunderkind imagemap.

wunderkind takes an argument object with the following keys:
width
integer
You must explicitly specify the width of the created div (fixes issue where if wunderkind is called on a hidden div the created div has no dimensions).
height
integer
You must explicitly specify the height of the created div (fixes issue where if wunderkind is called on a hidden div the created div has no dimensions).
hotspots
array
An associative array of hotspots.

Defining hotspots

The hotspots array takes the following key/value pairs:

name
string
Identifies a hotspot. Multiple hotspots can have the same name. If multiple hotspots share a name, hovering over one will activate all of the hotspots.
image
string
A URL (relative or absolute) to the image with the activated state for the hotspot.
top
integer
The y offset relative from the parent div that the hotspot begins.
left
integer
The x offset relative from the parent div that the hotspot begins.
width
integer
The width of the hotspot.
height
integer
The height of the hotspot.
onClick
Can be a:
  • string
    A URL to open on click of the hotspot.
  • function
    A function to execute on click. The function has one argument that will receive the event. In the event is data object called hotspot that provides access to the jQuery object that was clicked.

Example code

$('#insert-here').wunderkind({
	height : 480
  , width : 640
  , hotspots : [
	  { name    : 'black-kitty'
	  , image    : 'images/black_on.jpg'
	  , top     : 25
	  , left    : 87
	  , width   : 232
	  , height  : 313
	  , onClick : 'http://www.google.com/'
	  }
	, { name    : 'tabby-kitty'
	  , image    : 'images/tabby_on.jpg'
	  , top     : 35
	  , left    : 212
	  , width   : 428
	  , height  : 390
	  , onClick : function(e) {
			alert('clicked ' + e.data.hotspot.attr('rel')); 
		}
	  }
	]
});
